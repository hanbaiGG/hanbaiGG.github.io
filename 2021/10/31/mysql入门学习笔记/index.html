CREATE 操作

创建模式

CREATE SCHEMA<模式名>AUTHORIZATION<用户名>

定义表

CREATE TABLE TAB1( COL1 SMALLINT
COL2 INT
COL3 CHAR(20)
COL4 NUMERIC(10,3)
COL5 DECIMAL(5,2)
)

定义列与完整性约束条件

COL1 INT PRIMARY KEY

[列名][数据类型][完整性约束条件]

NOT NULL：非空约束C

指定的列不允许为空值

报错

UNIQUE：唯一约束U

指定的列中没有重复值，或该表中每一个值或者每一组值都将是唯一的

报错

PRIMARY KEY：主键约束P

唯一的标识出表的每一行,且不允许空值,一个表只能有一个主键约束

报错

FOREIGN KEY：外键约束R

一个表中的列引用了其它表中的列，使得存在依赖关系，可以指向引用自身的列

报错

CHECK：条件约束C

指定该列是否满足某个条件
