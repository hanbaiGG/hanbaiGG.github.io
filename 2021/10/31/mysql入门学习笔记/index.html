发表于 2021-10-31

CREATE 操作

创建模式

CREATE SCHEMA<模式名>AUTHORIZATION<用户名>

定义表

CREATE TABLE TAB1( COL1 SMALLINT
COL2 INT
COL3 CHAR(20)
COL4 NUMERIC(10,3)
COL5 DECIMAL(5,2)
)

定义列与完整性约束条件

COL1 INT PRIMARY KEY

[列名][数据类型][完整性约束条件]

完整性约束条件的类型

NOT NULL：非空约束C

指定的列不允许为空值

报错

UNIQUE：唯一约束U

指定的列中没有重复值，或该表中每一个值或者每一组值都将是唯一的

报错

PRIMARY KEY：主键约束P

唯一的标识出表的每一行,且不允许空值,一个表只能有一个主键约束

# DROP 操作
1.删除模式
DROP SCHEMA<模式名><CASCADE|RESTRICT>“二选一”
    CASCADE完全删除
    RESTRICT只删除空模式
